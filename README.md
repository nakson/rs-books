# Book Recommendation 

### How to Run

1.  Install environment

    ```shell
    cd ./src
    pip install -r requirements.txt
    ```

2.  Run the code

    ```shell
    uvicorn main:app
    ```

3.  Open webpage from Chrome

    - [Web Server for Chrome](https://chrome.google.com/webstore/detail/web-server-for-chrome/ofhbbkphhbklhfoeikjpcbhemlocgigb?hl=en)
